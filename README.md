# Automated Report Distribution Framework

## Overview

This project demonstrates how operational reporting workflows can be automated using Microsoft Power Automate, Excel Office Scripts, and OneDrive.

The framework was designed to reduce repetitive reporting tasks by automating report processing, file management, and email distribution.

## Technologies Used

- Microsoft Power Automate
- Excel Office Scripts
- Microsoft OneDrive
- Excel

## Features

- Automated report processing
- Automated file organization
- Automated email distribution
- Reusable workflow architecture
- Standardized reporting framework

## Workflow

Source Data
→ Office Script Processing
→ OneDrive Storage
→ Power Automate Trigger
→ Automated Email Distribution

## Business Impact

- Reduced manual reporting effort
- Improved reporting consistency
- Reduced repetitive tasks
- Created a scalable reporting process
